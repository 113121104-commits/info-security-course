## 作業1 - 目標/程序
1. 安裝 **VS Code** 並熟悉基本操作，學會連結 **GitHub 帳號**。  
2. 在本機建立一個資料夾，並用 VS Code 指定此資料夾為工作目錄。  
3. 將該資料夾初始化為 **Git 倉庫**，並推送到 GitHub，完成綁定。  
4. 在 VS Code 編輯 **MarkDown 檔案**，完成後推送到 GitHub，將檔案的 **GitHub 連結** 提供給老師。
